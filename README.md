# Nebula Code Atlas 🦀✨

A zero-dependency Rust CLI that turns any codebase into an offline visual software galaxy with stats, risk scanning, search, and dependency hints.

## Features

- Offline HTML dashboard
- Visual SVG codebase galaxy
- Language statistics
- File size and line analysis
- TODO / FIXME / HACK detection
- Risk scanning
- Search and filtering
- Zero dependencies
- Cross-platform

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/nebula-code-atlas.git
cd nebula-code-atlas
cargo build --release
```

## Usage

```bash
cargo run --release -- . atlas.html
```

Open atlas.html in your browser.
